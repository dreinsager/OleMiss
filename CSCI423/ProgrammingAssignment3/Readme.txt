This program executes the minimum, average, and maximum values when entering
multiple integers using pthread.

1. To compile the program run 'gcc Program3.c -lpthread'

2. To execute './a.out <add multiple integer>

For Example is inputing ./a.out 90 81 78 95 79 72 85,
the result will give 

the average value is 82
the minimum value is 72
the maximum value is 95

If no integer is given it will prompt an error saying to enter at least
one integer
